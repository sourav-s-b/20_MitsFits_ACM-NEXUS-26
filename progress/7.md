# NexusPath Progress — 01:15

## Bug Fixes: Dual-Path UI Rendering & Infinite Reroute Glitch
Resolved advanced edge-case bugs pertaining to the frontend map canvas failing to visualize alternative shadow networks, and backend WebSockets streaming stale data.

### 1. TomTom Shadow Path Separation
- `orchestration_routes.py`: Modified the backend's TomTom fetch logic. Previously, forcing `maxAlternatives=1` often just returned the exact same path the truck was already on. When drawing the map, it physically overlapped the main path making it invisible. **Fixed:** Upped `maxAlternatives=2` and parsed both arrays, explicitly forcing true topological alternatives that branch off visually.

### 2. The Infinite Reroute Glitch (Stale WebSockets)
- `decision_routes.py`: Clicking "Confirm Reroute" in the UI successfully updated the Database, but the backend forgot to wipe the `reroute_options` array from the `shipment` dictionary. Since WebSockets broadcast 20 times a minute, the UI instantly re-received the old options and rebuilt the "Reroute" warning UI. **Fixed:** Explicitly set `shipment["reroute_options"] = []` upon confirmation to kill the stale render cycle.
